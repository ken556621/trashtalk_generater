# 幹話產生器


## Features
1. 選擇職業
2. 產生幹話

## Installation
1. Github clone

    `git clone https://github.com/ken556621/trashtalk_generater.git`
    
2. Use terminal open file 'trashtalk_generater'

     `cd trashtalk_generater`
     
3. NPM install

    `npm install`
    
4. NPM run start

    `npm run start`
    
5. Open browser type

    `localhost:3000`
